
# New Year Note App

A minimal interface that allows us to leave the new year notes.


## Technologies and Libraries

- React Js
- Redux Tool Kit
- Tailwind CSS
- React Icons
- Firebase
- React Hot Toast
- Moment Js
- React Snow Effect
- Christmas Design



  
## Installation 

```bash 
    git clone https://github.com/mustafakaracuha/new-year-note-app/
```
```bash 
    cd new-year-note-app
```
```bash 
    npm install new-year-note-app
```
```bash 
    npm start
```
