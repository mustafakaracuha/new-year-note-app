
# New Year Note App

A minimal interface that allows us to leave notes.


## Technologies and Libraries

- React Js
- Redux Tool Kit
- Tailwind CSS
- React Icons
- Firebase
- React Snow Effect
- Christmas Design



  
## Install 

```bash 
  git clone https://github.com/mustafakaracuha/new-year-note-app/
  cd new-year-note-app
  npm install new-year-note-app
  npm start
```
